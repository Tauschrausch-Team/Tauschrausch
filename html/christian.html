Befehle für Informatik

Git-Befehle

  • git clone: Kopie des Gits auf eigenem Rechner speichern
  • git fetch: Informationen über das Git herunterladen
  • git pull: Gitdateien herunterladen

(hier wird gearbeitet!)

  • git commit: Änderung bestätigen und kommentieren
    • git commit -a: bezogen auf alles, sonst Datei angeben
  • git push: Gitdateien hochladen
  • git checkout: Branch (Namen hinter Befehl angeben) wechseln

Server starten

  • node server.js: Server starten
